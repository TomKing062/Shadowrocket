# 简介
Shadowrocket 的配置文件，每周四自动更新。
## 配置主文件链接
```
https://raw.githubusercontent.com/TomKing062/Shadowrocket/master/Shadowrocket.conf
```
### 使用建议（可选）
下载主配置后，点击`Shadowrocket.conf`，选择`扩展配置`，然后在新生成的配置中开启`HTTPS解密`(即MITM功能)并导入证书
